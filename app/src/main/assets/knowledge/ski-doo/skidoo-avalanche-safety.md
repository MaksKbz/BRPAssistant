# Ski-Doo — Лавинная безопасность

## Обязательное снаряжение
- **Лавинный датчик (бипер/ARTVA)**: носить на теле под верхней одеждой, режим TX
- **Лавинный зонд**: для определения точного местоположения засыпанного
- **Лавинная лопата**: жёсткая, не складная щётка
- **Лавинный рюкзак с ABS**: рекомендуется для горных маршрутов

## Оценка лавинной опасности
- Следить за официальным лавинным прогнозом (1–5 по шкале)
- Опасность 3 и выше — только опытные райдеры с полным снаряжением
- Признаки нестабильного снега: «буханье» при езде, недавние лавины, трещины

## Тактика на склоне
1. Один человек на склоне, остальные наблюдают снизу
2. Маршрут выхода — знать до начала спуска/подъёма
3. При высокой опасности — не пересекать конусы выноса лавин
4. Снегоход бросить при сходе лавины — уйти в сторону

## При засыпании снегоходчика
1. Переключить биперы в режим поиска (RX)
2. Определить направление сигнала
3. Зондировать на расстоянии сигнала
4. Быстро откопать голову и грудь
5. Вызвать спасателей сразу

## Нельзя
- ❌ Ехать по лавиноопасному склону в одиночку
- ❌ Снимать бипер при катании
- ❌ Игнорировать лавинный прогноз
