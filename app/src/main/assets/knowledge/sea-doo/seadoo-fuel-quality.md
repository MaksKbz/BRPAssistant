# Sea-Doo — Качество топлива и детонация

## Симптомы плохого топлива
- Двигатель работает жёстко и неровно
- Есть детонация под нагрузкой
- После заправки появилась ошибка двигателя
- Падает максимальная скорость

## Рекомендации
- Использовать минимум АИ-92, для 300-сильных supercharged моделей — лучше АИ-95
- Не хранить старый бензин весь межсезон без стабилизатора
- Избегать топлива с высоким содержанием этанола

## Если уже залили плохое топливо
- Не нагружать мотор
- По возможности разбавить качественным бензином
- Добавить очиститель топливной системы
- При выраженной детонации прекратить эксплуатацию
