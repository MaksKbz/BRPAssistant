# Can-Am — Справочник кодов ошибок

## Как читать коды ошибок
- На дисплее: мигающий символ + код E###
- Для чтения без сканера: зажмите кнопку MODE при включении (моргание индикаторов)
- Полная диагностика: BRP diagnostic tool (BUDS2) у дилера

## Двигатель (E01x – E03x)
| Код | Описание | Срочность |
|---|---|---|
| E011 | MAP-датчик (давление впуска) — нет сигнала | Средняя |
| E012 | MAP-датчик — сигнал вне диапазона | Средняя |
| E013 | TPS датчик — нет сигнала | Высокая |
| E014 | TPS датчик — сигнал вне диапазона | Высокая |
| E021 | Температура ОЖ — высокая | Высокая |
| E022 | Температура ОЖ — критическая | Критично |
| E031 | Давление масла — низкое | Критично |
| E032 | Температура масла — высокая | Высокая |

## Топливная система (E04x)
| Код | Описание | Срочность |
|---|---|---|
| E041 | Форсунка — обрыв/КЗ | Высокая |
| E043 | Топливный насос — нет давления | Высокая |
| E044 | TPS — калибровка потеряна | Средняя |
| E046 | MAP-датчик — неисправен | Средняя |
| E047 | IAT-датчик — неисправен | Низкая |

## Зажигание (E05x)
| Код | Описание | Срочность |
|---|---|---|
| E051 | Первичная цепь катушки — обрыв | Высокая |
| E052 | Вторичная цепь катушки — КЗ | Высокая |
| E055 | Датчик положения коленвала — нет сигнала | Критично |

## Трансмиссия (E06x – E07x)
| Код | Описание | Срочность |
|---|---|---|
| E061 | Датчик скорости переднего колеса | Средняя |
| E062 | Датчик скорости заднего колеса | Средняя |
| E071 | Соленоид 4WD | Средняя |
| E072 | Соленоид блокировки дифференциала | Средняя |

## Электрика (E08x – E09x)
| Код | Описание | Срочность |
|---|---|---|
| E081 | Напряжение АКБ — низкое | Средняя |
| E082 | Зарядка — нет зарядки | Средняя |
| E083 | Напряжение АКБ — перезаряд | Средняя |

## Действия при критических кодах
- **E022, E031**: немедленная остановка, глушить двигатель
- **E013, E014, E055**: ехать нельзя, требуется эвакуация или диагностика
- **E021, E032, E081**: снизить нагрузку, вернуться базу
- **Все остальные**: можно доехать до базы аккуратно, записать код
