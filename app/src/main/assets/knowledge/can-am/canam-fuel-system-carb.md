# Can-Am Outlander (карбюраторные модели) — Топливная система

## Симптомы проблем с карбюратором
- Не заводится после зимнего хранения
- Провал при резком открытии газа
- Нестабильные обороты на холостом ходу
- Переобогащение (чёрный дым, чёрные свечи)

## Причины
- Лак и смола в жиклёрах после хранения с топливом
- Забитый топливный фильтр
- Изношенная игла дросселя
- Мембрана поплавка с трещиной

## Что делать
1. Слить старое топливо из поплавковой камеры через дренажный винт
2. Промыть карбюратор чистящим спреем
3. Прочистить жиклёры
4. Проверить и при необходимости заменить топливный фильтр

## Хранение
- Перед сезонным хранением добавить стабилизатор топлива
- Предпочтительно — слить карб полностью через дренаж
