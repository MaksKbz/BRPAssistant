# Can-Am Maverick X3 — Подвеска (Front/Rear Double-A Arm)

## Характеристики подвески X3
| Параметр | Значение |
|---|---|
| Ход передней подвески | 356 мм (14") |
| Ход задней подвески | 381 мм (15") |
| Передняя колея | 1651 мм (65") |
| Задняя колея | 1702 мм (67") |

## Типичные неисправности
1. Износ сайлентблоков рычагов (интенсивная бездорожная езда)
2. Пробои амортизаторов Fox/Walker Evans при экстремальной езде
3. Ослабление болтов крепления рычагов (вибрация на жёстком грунте)
4. Деформация рычагов при ударах о камни (визуальный осмотр)
5. Повреждение пыльников ШРУСов (обязательная проверка после грязи)

## Полевые проверки

### Шаг 1: Проверка рычагов на деформацию
- Осмотрите все 4 рычага (2 передних A-arm пары + 2 задних) на предмет изгибов
- Деформированный рычаг → немедленная замена (нарушение геометрии = потеря управляемости)

### Шаг 2: Амортизаторы
- Осмотрите штоки на течь масла
- Проверьте крепёж амортизаторов (болты верх/низ)
- Регулировка отбоя (если Fox): поворотная рукоятка внизу амортизатора

### Шаг 3: Пыльники ШРУСов
- 4 приводных вала (по одному на каждое колесо)
- Разрыв пыльника → ШРУС без смазки → разрушение за 30–80 км
- В поле: обмотайте изолентой или куском резины + хомут

## Моменты затяжки (X3)
| Элемент | Момент |
|---|---|
| Болт верхнего рычага к раме | 65 Н·м |
| Болт нижнего рычага к раме | 75 Н·м |
| Гайка шаровой опоры | 55 Н·м + шплинт |
| Болт амортизатора (верх) | 65 Н·м |
| Болт амортизатора (низ) | 55 Н·м |
| Гайка ступицы | 180 Н·м |

## Настройка высоты кузова (preload)
- Fox QS3 амортизаторы: 3 позиции жёсткости (мягкая/средняя/жёсткая)
- Для тяжёлого бездорожья — жёсткая позиция, для трассы — мягкая
- Настройка preload пружины: специальный ключ (в комплекте с машиной)
