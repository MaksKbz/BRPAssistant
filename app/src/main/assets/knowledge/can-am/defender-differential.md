# Can-Am Defender — Трансмиссия и редукторы

## Трансмиссия Defender
- **CVT**: Visco-Lok QE (передний) + блокировка заднего дифференциала
- **Режимы**: 2WD / 4WD (нажатием кнопки из кабины) / 4WD Lock
- **Передаточные числа**: H (высший) / L (низший) / N / R

## Симптомы и причины

### Шум в трансмиссии
1. Недостаточный уровень масла в редукторе
2. Износ шестерён (металлическая стружка в масле)
3. Поломка подшипника вала
4. Неправильное трансмиссионное масло

### Вибрация при движении
1. Дисбаланс карданного вала
2. Износ крестовин карданного вала
3. Повреждённый привод колеса (ШРУС)
4. Деформация колёсного диска

### 4WD не включается
1. Неисправность Visco-Lok (нет давления масла)
2. Обрыв кабеля или отказ соленоида
3. Неисправность ЭБУ трансмиссии

## Полевые проверки

### Проверка масла редукторов
- Остановите машину на горизонтальной поверхности
- Снимите заливную пробку (она же — пробка уровня)
- Масло должно быть у нижнего края отверстия
- Чёрное масло с металлической стружкой → немедленная замена и диагностика

### Проверка карданного вала
- Под нагрузкой на малой скорости: покачивание = крестовины
- Осмотрите крестовины на ржавчину и люфт
- Смажьте через пресс-маслёнки при наличии

## Коды ошибок трансмиссии
| Код | Описание |
|---|---|
| E061 | Неисправность датчика скорости переднего колеса |
| E062 | Неисправность датчика скорости заднего колеса |
| E071 | Ошибка соленоида 4WD |
| E072 | Ошибка соленоида блокировки дифференциала |

## Когда можно ехать, когда нет
- Шум при включении 4WD → можно ехать в 2WD до сервиса
- Громкий металлический удар в трансмиссии → стоп, эвакуация
- Потеря 4WD в мягком бездорожье → можно вернуться в 2WD
- Потеря 4WD в сложном рельефе → не продолжать, риск застрять
